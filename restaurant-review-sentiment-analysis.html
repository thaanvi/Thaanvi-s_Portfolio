Restaurant Review Sentiment Analysis System

Introduction

This system analyzes restaurant reviews and classifies them as positive, negative, or neutral using sentiment analysis. Insights are displayed through a user-friendly dashboard to help restaurant managers, owners, and diners understand customer feedback more easily.

User Manual Overview

The user manual provides clear, beginner-friendly instructions for installing, launching, and using the sentiment analysis system. It guides users step-by-step so they can load review data, run sentiment classification, and interpret dashboard outputs.

Project Purpose

This tool is designed to make review analysis accessible to restaurant stakeholders who want fast, actionable insights from customer feedback. It helps identify customer sentiment trends and surface areas for improvement.

Target Users

  • Restaurant managers
  • Restaurant owners
  • Customer experience teams
  • Diners who want to understand feedback trends
  • Hospitality students and analysts

Key Features

  • Review sentiment classification into positive, negative, or neutral categories.
  • Easy-to-follow installation and launch instructions.
  • Dashboard visualization of sentiment trends and insights.
  • User-friendly interface aimed at non-technical stakeholders.

Tech Stack

Python, NLP, Sentiment Analysis, Dashboard Design
