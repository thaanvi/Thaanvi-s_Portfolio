Visual Speech-Aware 3D Facial Reconstruction

This project builds speech-aware 3D facial reconstruction using a SPECTRE/DECA backbone, MobileNetV2 feature extraction, and temporal convolution for audio-video synchronization. It is trained on the LRS3 dataset and deployed on AWS EC2 with a Streamlit frontend.

Project Summary

The system reconstructs 3D facial geometry from video while incorporating speech cues to improve realism and expression accuracy. It uses deep learning to fuse audio and visual features, allowing improved facial detail and lip-sync performance in generated 3D outputs.

Key Features

  • Speech-aware 3D facial reconstruction using synced audio-video input.
  • SPECTRE/DECA backbone with MobileNetV2 for efficient feature extraction.
  • Temporal convolution to capture motion and expression across frames.
  • Deployment on AWS EC2 with a Streamlit user interface.

Tech Stack

Python, PyTorch, OpenCV, AWS, Streamlit
